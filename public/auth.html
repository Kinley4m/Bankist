Real Authentication

Login / Signup with Node.js + MongoDB

This page is connected with the real backend API. Signup creates a new account, login verifies your password, and a JWT token is stored in the browser.

Integrated APIs

  • POST /api/auth/signup
  • POST /api/auth/login
  • GET /api/account/profile
  • POST /api/account/deposit
  • POST /api/account/withdraw
  • POST /api/account/transfer
  • POST /api/account/loan
